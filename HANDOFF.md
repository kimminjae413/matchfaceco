## ✅ 종결: 배출계수 출처 (2026-07-13 리서치로 확정)

**`0.56 kgCO₂e/kWh` 는 베트남 정부 공표 문서 어디에도 없는 값이었다. 다시 쓰지 말 것.**

- 베트남 공식 전력망 계수는 천연자원환경부 **기후변화국(Cục Biến đổi khí hậu, DCC)** 이 매년 **공문(Công văn)** 으로 공표한다.
  - 2023년(최신 확정) **0.6592 tCO₂/MWh** — 공문 **1726/BĐKH-PTCBT** (2024 발표)
  - 2022년 0.6766 — 공문 327/BĐKH-PTCBT
  - 2021년 0.7221 / 2020년 0.8041 / 2019년 0.8458
- **MONRE 결정 4432/QĐ-BTNMT(2022)** 는 배출계수 *목록*을 공표하는 결정이지 그리드 계수 숫자를 담은 문서가 아니다. **통람 38/2023/TT-BCT** 는 산업부문 MRV *절차* 규정이라 숫자가 없다. 둘 중 어느 것도 "0.56의 출처"로 인용하면 틀린다.
- 0.56은 국제금융기구(IFI) 프로젝트 파이낸싱 심사용 default grid factor 데이터셋 값으로 추정된다(2차 출처 기반, 1차 미대조). 정부 공식치보다 **15~20% 낮아** 배출량을 실제보다 작게 보이게 만든다 — 그린워싱 지적의 전형.
- → Formula 섹션에 **DCC 공문 1726 / 0.6592** 를 출처와 함께 명시함.

## 이 브랜치에서 한 일

### P0 — 신뢰를 깎던 사실관계
- **제조사 수 불일치 통일**: 히어로만 32곳, 나머지는 30곳+ 였음 → 문서 기준 **30곳+** 로 단일화
- **없는 실적 제거**: 대시보드 누적 카운트 → 기능 상태 표기 + "예시 화면" 명시 (☝️ 결정 대기 2번)
- **근거 없는 "36건 제조 아웃소싱 운영 경험" 삭제** — 사업계획서·서비스소개·등록증 어디에도 없음
- **배출계수 출처 정직화** (☝️ 결정 대기 3번)
- **푸터 주소를 사업자등록증 표기로 정정** (건물명 포함)

### P1 — AI 랜딩 클리셰 제거
- 히어로 **터미널(신호등 점 3개 + 타이핑) → PCF 산정서 문서**로 교체. 이 제품 구매자는 구매·소싱·ESG 담당자지 개발자가 아니다 — 터미널은 애초에 장르 오류였음
- **CinematicBand(나뭇잎·새 영상) 통째 삭제** — CBAM 규제 대응 상품에 감성 자연 영상은 그린워싱 신호
- 알약 배지 3종 / 방사형 글로우 / 격자 배경 / 스태거 fadeUp 제거
- 영어 대문자 모노 eyebrow(THE PROBLEM 등) → 한국어 라벨 + 괘선
- 헤드라인 폰트 **Wanted Sans** 도입, Quicksand 제거
- **다크 면을 근접 블랙(#05100C) → BI 딥 포레스트(#083630)로.** "근접 블랙 + 단일 애시드 그린"이 현재 가장 흔한 AI 랜딩 룩이라, 그린(BI라 못 버림)은 두고 바닥을 올려 축을 이동
- VisualBand의 **AI 생성 공장 작업자 목업 사진 제거** (실사 확보 시 재투입)

### 레이아웃 결함
- Features 벤토 그리드 깨짐 (6칸에 4+2/2+2 → 마지막 2칸 공백) → 번호 목록으로 재구성
- 비교표 강조 열이 고정 내비 아래로 파고들어 헤더 잘리던 문제

### 전환
- `mailto:` 링크뿐이던 CTA에 **구조화된 문의 폼**(회사·담당자·공정·수량) + **카카오 채널** 추가
- 서버 없는 정적 호스팅이라 입력값을 메일 본문으로 조립하는 방식. 나중에 Formspree/Tally 엔드포인트로 교체만 하면 됨

## 남은 작업 (우선순위)

1. 푸터의 영어 라벨(`PRODUCT` / `COMPANY`) → 한국어
2. **샘플 리포트 PDF** — "어떻게 계산하나요"를 눌러도 설명만 나옴. 구매자가 진짜 원하는 건 실제 산정서 한 장. 설명 열 줄보다 강하다
3. **팀·대표 소개** — 초기 B2B 신뢰의 절반은 "누가 하는가"
4. **개인정보처리방침·이용약관** — 문의 폼 받으려면 법적으로 필요
5. **영문 버전** — 베트남 제조사를 파트너로 모집하는데 한국어만 있음
6. `matchfac_eco_서비스소개.md`의 **옛 주소(대덕구 동춘당로 83) 수정** — 여기서 복사해 IR·제안서 만들면 틀린 주소가 계속 퍼진다
